What is Bootstrap? Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites. Bootstrap 5 is the newest version of Bootstrap.
Quickly design and customize responsive mobile-first sites with Bootstrap, the world’s most popular front-end open source toolkit, featuring Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful JavaScript plugins.

Most Benefits of Bootstrap.
  1. Across devices, across browsers: Compatible with all modern browsers.
  2. Responsive Layout:Bootstrap provides a responsive, mobile-first streaming grid system. It can adjust the page to the user's screen size to make it look good on all sizes.
  3. CSS modular:Many CSS classes are pre-defined in bootstrap. When using the class, you can directly assign the corresponding class name, such as text-left,text-align,.table, etc
  4. Built-in JavaScript plug-in:Bootstrap relies on JQuery because it provides a variety of useful JQuery plugins that allow developers to implement common effects on the WEB.
  5. Rich components:Bootstrap provides a variety of useful components, including navigation, tabs, toolbars, buttons, etc.

How to use
Accordion
Build vertically collapsing accordions in combination with our Collapse JavaScript plugin.
The accordion uses collapse internally to make it collapsible. To render an accordion that’s expanded, add the .open class on the .accordion.
Collapse
Toggle the visibility of content across your project with a few classes and our JavaScript plugins.
The collapse JavaScript plugin is used to show and hide content. Buttons or anchors are used as triggers that are mapped to specific elements you toggle. Collapsing an element will animate the height from its current value to 0. Given how CSS handles animations, you cannot use padding on a .collapse element. Instead, use the class as an independent wrapping element.
List group
List groups are a flexible and powerful component for displaying a series of content. Modify and extend them to support just about any content within.
Disabled items Add .disabled to a .list-group-item to make it appear disabled. Note that some elements with .disabled will also require custom JavaScript to fully disable their click events (e.g., links).
